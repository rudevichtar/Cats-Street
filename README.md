# Cat Street

## Запуск проекта

Для проверки работоспособности программного продукта подготовлена готовая сборка проекта.

Путь к сборке:

Build/

Исполняемый файл:

Build/Cat Street.exe

Для запуска необходимо открыть файл `Cat Street.exe`. Все вспомогательные файлы и каталоги сборки должны оставаться в исходной структуре относительно исполняемого файла.

## Описание проекта

**Cat Street** — это игровой симулятор, разработанный на движке Unity с использованием языка программирования C#. Игрок управляет городской средой и помогает кошкам безопасно перемещаться между различными объектами городской инфраструктуры.

Основная задача игрока заключается в обеспечении благополучия животных, удовлетворении их потребностей и предотвращении возникновения опасных ситуаций. Для достижения целей используются механики размещения ресурсов, управления дорогами и контроля опасностей.

---

## Основные возможности

* система передвижения кошек;
* система потребностей (голод, сонливость, настроение, здоровье);
* поиск пути между объектами городской среды с использованием алгоритма A*;
* динамическая блокировка участков маршрута;
* размещение ресурсов (миски с едой и лежанки);
* система дорожного движения;
* управление светофорами;
* генерация опасностей в виде собак;
* система игровых дней и смены времени суток;
* сохранение и загрузка игрового состояния в формате JSON;
* интерфейс управления игровым процессом.

## Архитектура проекта

Проект построен на основе модульной архитектуры и включает следующие основные подсистемы:

* модуль управления кошками;
* модуль графа городской среды;
* модуль поиска пути;
* модуль потребностей;
* модуль ресурсов;
* модуль дорожного движения;
* модуль светофоров;
* модуль опасностей;
* модуль интерфейса;
* модуль сохранений;
* модуль игрового цикла;
* модуль главного меню.

---

## Алгоритмы

В проекте используются следующие алгоритмы:

* алгоритм поиска пути A*;
* конечный автомат состояний (FSM) для принятия решений кошками;
* алгоритм динамического перепостроения маршрута;
* алгоритм сохранения и загрузки данных;
* алгоритм управления игровым временем.

---

## Сохранение данных

Состояние игры сохраняется в формате JSON и включает:

* параметры игрового мира;
* информацию о кошках;
* размещённые ресурсы;
* светофоры;
* опасности;
* заблокированные ноды графа.

---

## Цель проекта

Создание интерактивного симулятора городской среды, позволяющего изучать алгоритмы поиска пути, поведение игровых агентов, управление динамически изменяемой графовой структурой в игровой форме, а также оперативное принятие решений.
